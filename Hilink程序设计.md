# 程序设计说明
# 对于硬件系统的设计原则一定是简化到不能再简
## 阶段零：启动和系统初始化（Boot & Init）

两端设备（Hilink 和 Hisense）上电后各自执行的动作。

- **基础外设初始化**：初始化 GPIO、UART（TDMA 与 GPS/4G）、IIC（OLED 与 NFC）
- **定时器初始化**：配置 TIM2 为向上计数，频率设为 10kHz（即 0.1ms 精度），但不启动
- **中断优先级配置（NVIC 独裁）**：
  - 抢占优先级 0（最高）：1PPS 脉冲中断、双触点开关中断 (Hilink)、光电门中断 (Hisense)
  - 抢占优先级 1：TDMA 串口接收中断
  - 抢占优先级 2（最低）：4G/GPS 串口解析等其他任务
- **时间引擎对齐（等待 GPS 锁定）**：
  - 等待 GPS 模块抓取卫星
  - 硬件中断：当 1PPS 上升沿到来，触发 EXTI，瞬间将 `TIM2->CNT = 0` 并启动计时
  - 软件解析：在主循环解析 NMEA 报文，获取绝对秒数 unix_sec
  - 当上述两步完成，标记 `sync_flag = 2`，系统准备就绪，OLED 显示「系统就绪」

---

## 阶段一：检录准备（Hilink 主循环）

运动员在山顶准备出发。

- **NFC 轮询**：Hilink 的主循环通过 IIC 不断读取 NFC 模块
- **身分绑定**：读取到选手手环 ID（例如：ID_001）
- **入队操作**：
  - 生成一个出发序号 Seq = 1
  - 将该选手数据建立一个结构体，推入 Hilink 内部的环形 FIFO 队列的尾部（tail）
- **OLED 屏幕**通过 IIC 更新显示：「下一位出发：ID_001」

---

## 阶段二：起跑触发（Hilink 中断优先）

运动员推开起跑门的瞬间。

- **瞬间获取（EXTI 0）**：运动员推开双触点开关，触发 Hilink 的最高优先级中断
  - 中断函数内：立即读取当前时间戳 `T_start = unix_sec + TIM2->CNT`
  - 将这个 T_start 更新到 FIFO 队列中刚刚检录的那位选手的数据
  - 退出中断（耗时仅几微秒）
- **状态更新（主循环）**：
  - 点亮双色 LED 为绿色，蜂鸣器响起
  - OLED 更新显示：「ID_001 已出发，滑行中...」

---

## 阶段三：终点冲线（Hisense 中断 → TDMA 传输）

运动员到达山下，切断光电门。

- **在 EXTI 0**：光电传感器触发 Hisense 端的最高优先级中断
  - 中断函数内：立刻读取当前的绝对时间戳 `T_end = unix_sec + TIM2->CNT`
  - 创建自增的冲线序列号 End_Seq
  - 退出中断
- **无线发送（主循环）**：Hisense 通过 UART 控制 TDMA 模块，将数据包 `[End_Seq, T_end]` 发送给山顶的 Hilink
- Hisense 点亮 LED 提示冲线成功

---

## 阶段四：结算与上报（Hilink 主循环）

Hilink 收到终点数据，计算最终成绩。

- **接收终点数据**：Hilink 的 TDMA 串口收到数据包 `[End_Seq, T_end]`
- **FIFO 匹配和计算**：
  - Hilink 提取 FIFO 队列头部（head）的选手（即最早出发且仍在途的 ID_001）
  - 计算成绩：`Result = T_end - FIFO_Queue[head].T_start`
- **上报与出队**：
  - 透过 IIC 将成绩显示在 OLED 屏幕上
  - 透过 UART1(DMA模式)控制 4G 模块，将 JSON 格式的成绩数据（包含选手 ID、成绩）通过 MQTT 上报给云端 App
  - 将该选手从 FIFO 队列中踢出（head 指针后移），等待匹配下一位选手

---

## 阶段五：异常处理防错机制（Hilink 主循环/中断）

处理运动员摔倒退赛（DNF）的情况，防止 FIFO 崩溃。

### 机制 A：超时自动踢出（主循环）

- Hilink 的 `while(1)` 每次循环都会检查 FIFO 队列的 head 选手
- 如果 `当前绝对时间 - T_start > 180秒`（假设 3 分钟超时），则判定该选手未完赛
- 标记该选手为 DNF，上报云端，并将其移出队列（head++）

### 机制 B：裁判人工一键作废（EXTI 中断）

- 裁判得知有人摔倒，按下 Hilink 上的「拔码开关/按键」
- 触发 GPIO 中断，程序强行把 FIFO 队列的 head 选手踢出并标记为 DNF
- 确保下一位顺利冲线的选手不会匹配错成绩
